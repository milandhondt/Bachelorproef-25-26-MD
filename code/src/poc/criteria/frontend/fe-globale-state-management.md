Controleer op het correct toepassen van globaal state management.

Zoek actief naar het volgende in de React-codebase:

- Zoek naar het gebruik van globale state management oplossingen zoals React Context (`createContext`, `Provider`, `useContext`), Redux, Zustand, of overige oplossingen.
- Verifieer dat deze oplossingen effectief gebruikt worden.

Belangrijke extra vereisten:

- Vermeld in het bewijs expliciet óf er gebruik gemaakt wordt van globale state en welke technologie (Context, Redux, Zustand, etc.) hiervoor gekozen is.
- Vermeld waarvoor deze globale context(en) of stores worden gebruikt (bijv. authenticatiestatus, thema, centraal winkelwagentje).

Beoordeel: wordt er efficiënt gebruik gemaakt van globale state management waar nodig, en zijn de stores bedoeld voor de juiste data?
