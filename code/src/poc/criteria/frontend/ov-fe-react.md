Ontvankelijkheidscriterium (Front-end): De applicatie is gemaakt in React.

Controle-instructies:

- Open de `package.json` van de frontend-applicatie en verifieer de aanwezigheid van `react` en `react-dom` als dependencies.
- Open een bronbestand in `src/` en verifieer React-gebruik: JSX-syntax (`.tsx`/`.jsx`), `import React from 'react'` of `import { useState, useEffect } from 'react'`, en componentdefinities.
- Verifieer dat het project een herkenbare React-projectstructuur heeft: `src/`, `public/`, en een entry point (`main.tsx`, `index.tsx` of `App.tsx`).

Beslisstatus:

- aanwezig: React-gebruik is ondubbelzinnig aantoonbaar in dependencies én broncode.
- afwezig: React niet aanwezig of het project is in een ander framework gebouwd.
- onduidelijk: onvoldoende bewijs om React-gebruik te bevestigen.
