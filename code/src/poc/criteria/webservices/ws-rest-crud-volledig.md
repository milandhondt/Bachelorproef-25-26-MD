Controleer de REST API op volledige CRUD-operaties.

Zoek actief naar het volgende in de NestJS-codebase:

- Lees de controllerbestanden (`*.controller.ts`) en verifieer de aanwezigheid van alle standaard CRUD-operaties per entiteit: `@Post` (Create), `@Get` (Read — lijst én enkelvoud via id), `@Put` of `@Patch` (Update), `@Delete` (Delete).
- Controleer minimaal 2 entiteiten naast `User`.
- Een entiteit waarbij een of meerdere operaties volledig ontbreken telt als onvolledig.

Beoordeel: zijn alle CRUD-operaties aanwezig en correct gekoppeld aan HTTP-methoden voor meerdere entiteiten?
